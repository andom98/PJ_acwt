# 近未来SF戦記ゲーム コンセプト

## タイトル（仮）
Operation Straits: 2048

## 企画の狙い
- 1人開発で完成可能なスコープで、近未来SF戦記の緊張感を体験させる
- 小規模でも成立するコア体験に絞り、Steam公開を最短で目指す
- 価格は初回作の戦略として 980円を第一候補にする

## ジャンル
- シングルプレイ タクティカルアクション（短編キャンペーン型）
- 視点: 三人称（TPS）

## 世界観（2048年）
- 舞台: 島嶼・海峡戦域
- 各国は無人機群、衛星妨害、海底ケーブル攻撃を常態化
- 正規軍と民間インフラの境界が曖昧で、情報優勢が勝敗を左右

## コア体験（プレイヤーに感じてほしいこと）
- 見えない敵（ドローン/電子戦）に追い詰められる緊張
- 限られたリソースで任務優先度を決断する重さ
- 戦術的勝利と人道的損失のトレードオフ

## コアループ
1. 作戦ブリーフィングで目標選択
2. 現地潜入（索敵・攪乱・交戦）
3. 通信/電力/補給の確保または破壊
4. 撤収して戦果評価、次任務へ反映

## MVP（最初に完成させる範囲）
- マップ 1（離島基地+港湾エリア）
- プレイアブル任務 3
- 敵種 3（歩兵、軽車両、偵察ドローン）
- 武器カテゴリ 3（AR、DMR、爆薬）
- 最低限UI（ミニマップ、弾薬、目標表示）
- クリア時間: 2〜3時間

## 非MVP（後回し）
- 大規模ストーリー分岐
- マルチプレイ
- 複雑なクラフト/拠点建築
- フォトリアル最適化の追い込み

## 価格と販売の初期方針
- 第一候補: 980円
- 弱気案: 580円（短編特化の場合）
- 強気案: 1480円（ボリューム増加時のみ）
- まずは Steam Direct回収ライン突破を最初の現実目標に置く

## 成功指標（初期）
- 技術指標: 30fps以上を安定維持（最低環境）
- 開発指標: MVPの3任務をプレイ可能
- 事業指標: ウィッシュリスト100件、初月レビュー10件
